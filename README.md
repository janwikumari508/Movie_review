# 🎬 Movie Review Website

A modern and responsive movie review website built using HTML, CSS, and JavaScript.

## 📌 Overview

This project is a frontend web application that allows users to explore movies through a clean and visually appealing interface. The focus of the project is responsive design, user experience, and modern UI development.

## ✨ Features

- Responsive design
- Modern user interface
- Movie cards
- Search functionality
- Hero section
- Movie details section
- Smooth animations
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/janwikumari508/Movie_review.git
```

2. Open the project folder

3. Open `index.html` in your browser.
